## 9. Función exit
- Para por completo el programa

## 10. Sentencia goto
- Salto incondicional de un lugar a otro del programa
- El destino se marca mediante etiquetas seguidas por dos puntos
```
goto ETIQUETA;
...
ETIQUETA:
printf(...);
...
```
