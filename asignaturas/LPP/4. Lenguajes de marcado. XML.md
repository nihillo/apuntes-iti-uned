## 4.3 Modelado de datos en XML
- esquema
- DTD
- XSD (XML-Schema)
- diferencia DTD - XSD
	- DTD sintaxis específica, XSD sintaxis xml
	- XSD se puede manipular
	- XSD soporta tipos, DTD trata todo como cadenas
	- DTD sólo permite asociación entre un documento y su DTD
